Numbered Circles

The numbered circle and connecting line are automatic — to the first column, just add a paragraph with the step number.

e.g. HTML source in column 1

<p>1</p>

Continuation Row (subrow)

Adds more content underneath an existing step without a new number/circle — used to align the content in column 2 with the panel content in column 3.
  • Class attribute subrow is added to the row element, tr.
  • Column 1 is left blank.

e.g. HTML source for a continuation row (subrow)

<tr class="subrow"> ... </tr>

Lists

Use the list elements for lists.
  • Unordered list

  1. Ordered list, default (1, 2, 3, ...)
  1. Ordered list (A, B, C, ...)
Add the type attribute to the ol element to change the numbering style.

e.g. HTML source for lists


Ordered list type="A, B, C, ..."
<ol type="A"> ... </ol>
This can only be added to the ol element in HTML </> edit mode, not in the WYSIWYG editor.

Checklist

Use the checklist class on the ul element to create a checklist.
  • Checklist item 1
  • Checklist item 2

e.g. HTML source for checklist

<ul class="checklist"> ... </ul>
This can only be added to the class attribute if the whole list is selected.

Checklist Checkmark

Use the checklist checkmark class on the ul element to create a checklist with checkmarks.
  • Checklist item 1
  • Checklist item 2

e.g. HTML source for checklist with checkmarks

<ul class="checklist checkmark"> ... </ul>
This can only be added to the class attribute if the whole list is selected.

4

Blockquote

Used to add a box around a block of text, e.g. for a quote or reference.

Types of blockquote styles

  • Default blockquote
  • Blockquote with box
  • Blockquote with warn

Select the text and set the style to blockquote. Add the class attribute as needed, e.g. box or warn.

Attribute class can only be added to the class attribute if the whole block of text is selected.

e.g. HTML source for blockquote

"Default"

<blockquote> Quote text </blockquote>

Box

<blockquote class="box"> Box </blockquote>

Warn

<blockquote class="warn"> Warn </blockquote>
Warning Bold the "Warning" text to have icon left of title

<blockquote class="warn"> <b>Warning</b> Bold the text with icon title </blockquote>

Tables

Use the table element to create a table.

e.g. table

Header 1 Header 2
Row 1, Column 1 Row 1, Column 2
Row 2, Column 1 Row 2, Column 2
  • Optional: add attribute class left to the table element to left align the content
  • <table class="left"> ... </table>

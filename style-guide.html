Lists

Use the list editor tool.

Two types:

  • Unordered list
  • Ordered list
NOTE: for ordered lists using alphabetical letters:

Add the type attribute to the ol element to change the numbering style.

This can only be added to the ol element in HTML </> edit mode, not in the WYSIWYG editor.

  • Ordered list

Checklist

  • Checklist item 1
  • Checklist item 2

Checklist checkmarked

  1. Select the entire list.

  • Checklist item 1

Blockquote

Used to add a box around a block of text, e.g. for a quote for communication.

Types of blockquote styles

  • Blockquote (default)
  • Blockquote with box
  • Blockquote with warn

Select the text and set the style to Blockquote. Add the attribute class as needed, e.g. box or warn.

Note: Attribute class can only be seen if the whole block of text is selected.

e.g. HTML source for blockquote

"Default"
<blockquote> Quote text </blockquote>

Box
<blockquote class="box"> Box </blockquote>

Warn
<blockquote class="warn"> Warn </blockquote>

Warning Bold the "Warning" text to have icon left of title
<blockquote class="warn">
<b>Warning</b>
... </blockquote>

Tables

Use the table tool to create a table.

Header row

  1. Click on top row of the table
  2. Set Header row by selecting the table tool > Row > Row Properties > Row type

Left alignment

  1. Select the entire table.
  2. Add attribute class left to the table element to left align the content

e.g. table

Header 1 Header 2
Row 1, Column 1 Row 1, Column 2
Row 2, Column 1 Row 2, Column 2
  • Default: content aligned center
  • <table> <thead>...</thead> </table>

e.g. table with left alignment

Header 1 Header 2
Row 1, Column 1 Row 1, Column 2
Row 2, Column 1 Row 2, Column 2
  • <table class="left"> ... </table>

Printing

The page has style set up to print for A1 portrait size. Longer pages will flow to the next page.

In Learning Hub Manage Files > CPB Key Processes Posters folder, click on the file to download it to your computer. Open the file in a browser and print it.

Note: Recommended browser is Chrome. Firefox may not print correctly, will require scale adjustment.

How to print

  1. Download the file from Learning Hub Manage Files by going to Edit Files > Manage Files > CPB Key Processes Posters.
  2. Click the file name (i.e. 1.html,..etc.) to download the file to your computer.
  3. Open the file in a browser (i.e. Chrome).
  4. Print the file (i.e. in Chrome — right click in the page > print) using A1 paper size.
